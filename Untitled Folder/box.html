Box

Box is the working title for a Ruby On Rails web app for UT Arlington. It is a vote-able email repository, for staff in the Office of Information Technology. Intended to replace the scattered system of passing precomposed emails among staff through Word Documents, Box allows users to sign up, dump their precomposed emails into the repository, and the best are voted up by users within the group. The application has user accounts, as well as encrypted passwords and recovery through Devise. The app features a responsive layout for mobile access, and utilizes current CSS techniques such as box shadowing and external webfonts. Upcoming features involve the inclusion of Angular JS for live searching, group management, and login from Facebook and Google by OmniAuth integration.

The app is internally hosted, but I have recently put up a test instance on Heroku for demonstration purposes: http://pure-crag-1028.herokuapp.com/
