UTA OIT Help Desk Mobile Site

‹ ›

The UTA Help Desk is the primary source of IT support for the University of Texas at Arlington. The OIT Help Desk supports over 33,439 students and over 3,000 faculty and staff. Working together with another developer, I initiated the push to create a responsive website which would be accessible on tablets and smartphones, while being light and highly compatible with a large user base. The site is deployed and being tested for performance, after which the various other IT systems it ties into will be modified as well. The Help Desk mobile site is written using HTML, CSS and JavaScript within a jQuery Mobile framework. The site is currently integrated into Cascade CMS, however migration and expansion into a Ruby on Rails application on a Linux server is currently being pursued. The site is fully responsive to multiple device widths, and the use of jQuery mobile was chosen due to its compatibility with a large range of devices. The swipe feature at the top is a combination of swipe.js, and an open source project forked from its Github, Swipe 2.

On a mobile device, redirects from desktop site: www.uta.edu/helpdesk

Direct link: http://www.uta.edu/oit/cs/helpdesk/mobile.html
